This template contains the information regarding the input files that are to be uploded to run the Logic_Report Automation App

There are two files that needs to be given as input.
1. Data file --> This file is the input file that contains various internal and external data for the movies.
                 This file is generated by following the data preparation steps given in the repository.  
                  Sample file is uploaded as Raw_data.xlsx

2. Average Partner Wise Rating File ---> This file gives the information related to the Partner Rating of the movies.  
                                          Sample file is uploaded as Average_Partner_Wise_Rating.xlsx

Note:- The column names and column position has to be in the order given in the Sample files.
